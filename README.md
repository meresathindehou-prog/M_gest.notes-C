## 🚀 🔍 Aperçu

Le **Gestionnaire de Notes en C** est une application console permettant de :

✔ gérer une liste de notes  
✔ effectuer des calculs statistiques  
✔ sauvegarder et charger les données  

Ce projet met en pratique les fondamentaux du développement logiciel en C avec une approche modulaire et structurée.

---

## 🎯 Objectifs

- Renforcer les bases en programmation C  
- Comprendre les pointeurs et structures de données  
- Manipuler les fichiers  
- Créer une application interactive

### 📌 Gestion

- 🔝 Note maximale
